zooniverse clone

Cloned zooniverse website

Zooniverse.org is a non profit organisation that allows people with different skills to showcase their skills, and inspire others to do extraordinary things. Speaking from a developer perspective, I found zooniverse website fancy and a good work for an upcoming developer to replicate.

After, the blog page that i worked on using react, I felt the need to push my react capabilities further by working on something big and complex. I initially wanted to replicate only the home page but with time I coninued adding more pages and linking them to the router. This project helped to solidify my knowledge of reusable components. I intended to build the entire website
