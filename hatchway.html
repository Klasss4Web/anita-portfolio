Hatchway Students Page

Hatchway Students Record

Hathcway students page is a website for a fictitious school, Hatchway.It is meant to display students information including grades. It was built using React and Bootstrap.

This single page website represent a fictitious school with fictitious students. The aim of the exercise was to test and keep my API fetching skills alive while working with React.

Getting the students data from the api and rendering them on the user interface was a bit easy but the real work came when I needed to interact with the API from the user interface. There was no provision for me to make a post request to the api, hence I have to look for a way to manipulate the data and create the logic from the front end. That was a bit difficult considering the fact that a provision for post requests would have made it easier to deal with from the front end. The completed app involved some fetching data using fetch, manipulating some of those data using javascript array methods like map, filter and reduce.
